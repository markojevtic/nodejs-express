###### Express Starter

* use starter to create project

```jshelllanguage
$ npm install express-generator -g
$ express --no-view myapp
   create : myapp\
   ...
   create : myapp\bin\www
   
   change directory:
     $ cd myapp
     
   install dependencies:
     $ npm install
     
   run the app:
     $ DEBUG=myapp:* npm start 
```
